Stop tailing Claude's session log.

A VSCode extension that surfaces what Claude Code is doing — token burn, cost, system health, memory — without leaving your editor.

View on GitHub What it does

Available on the VSCode Marketplace. Read-only, 100% local, zero telemetry.

Live token + cost rollups

Input, output, cache read, cache write — broken out by model family, rolled up session / day / week / month / year. Budget caps with progress bars.

20 tabs, fully composable

Now, Mac, Watchtower, Agents, Routines, Timeline, Library, Settings, Security, Talk, and more. Each tab shows a line icon. Pick which tabs are visible and which widgets appear on each.

Mac system stats

CPU (used %, core count, model), Memory (pressure, wired/active/compressed/free), Energy (battery %, cycle count, health), Disk (per-volume bars), Network (rx/tx KB/s, SSID, IPs). Green/amber/red tones match existing thresholds.

Memory, skills, agents

Your MEMORY.md entries, skill palette, and agent definitions from ~/.claude/agents/ — all inline. Pinnable memory, prompt library, global search across every session JSONL.

Privacy — 100% local

Claude Code already touches everything sensitive on your machine. A tool that surfaces its state has no business phoning home.

  • No network calls. Zero fetch, http, or XMLHttpRequest in the source. Grep it yourself.
  • No telemetry, no analytics, no crash reporting. Ever.
  • Read-only. Phase 1 never writes to ~/.claude/. Only readFileSync, readdirSync, statSync, watch.
  • Bounded reads. Only ~/.claude/projects/<encoded-cwd>/ and its memory/MEMORY.md.
  • Sandboxed webview. Strict CSP: default-src 'none'; connect-src 'none'; form-action 'none'.
  • Zero runtime dependencies. No supply chain to audit at runtime.

Open source. Read the full privacy policy or audit the source.

Why

Claude Code stores rich state on disk: session JSONL, persistent memory, hooks, MCP config. The CLI surfaces almost none of it. Cockpit reads what's already there and gives you a passive HUD instead of tail -f.

It's read-only by design — Claude Code is the source of truth. Cockpit just makes the truth visible.

Install

code --install-extension dashable.claude-code-cockpit

Or search "Claude Code Cockpit" in the VSCode Extensions sidebar.

To build from source: clone the repo, run npm install && npm run compile, then press F5 in VSCode.
